# BimiPal

**BimiPal** — це Telegram-бот для автоматизованого ведення спільного сімейного бюджету. Проєкт забезпечує повний фінансовий контроль, інтегруючи зручний інтерфейс із потужною аналітикою на базі Google Sheets.

### Ключові можливості:

* **Фінансові спринти:** Робота з динамічними бюджетами замість жорстких місячних лімітів.
* **Швидкий ввід:** Миттєва фіксація витрат через інтуїтивне підменю з можливістю швидкого редагування суми.
* **Аналітика та звіти:** Автоматична генерація PDF-звітів та кругових діаграм витрат.
* **Розумний контроль:** Щоденний розрахунок "денної норми" (Burn Rate) та автоматичні нагадування про регулярні платежі.
* **Сімейна синхронізація:** Роздільні баланси для користувачів та єдиний облік спільного капіталу.

### Технічний стек:

* **Мова:** Python 3.12+ (aiogram 3.x, асинхронність)
* **Backend:** Google Sheets API (gspread)
* **Аналітика:** Matplotlib, ReportLab
* **Інфраструктура:** Docker (локальний сервер 24/7).
