## 📂 BIMIPAL PROJECT CARD (Context for AI)

**Опис:**
Telegram-бот для ведення сімейного бюджету (Вадим та Аня). Дані зберігаються в Google Sheets. Проєкт розрахований на роботу 24/7 на робочому ПК (Windows), розробка ведеться на ноутбуці.

### 🛠 Технічний стек
* **Мова:** Python 3.x
* **Бібліотеки:**
    * `aiogram 3.x` (асинхронний бот)
    * `gspread` (робота з таблицями)
    * `oauth2client` (авторизація Google)
    * `matplotlib` (генерація діаграм)
    * `apscheduler` (планувальник завдань/нагадувань)
    * `re` (RegEx для очищення даних та валют)
* **База даних:** Google Sheets (Service Account).
* **Хостинг:** Локальний робочий ПК (Windows 10/11) у режимі 24/7.
* **Версійність:** Git (GitHub Private Repo).

---

### 🗂 Структура файлів
* `main.py` — Основний файл. FSM, хендлери (Витрати, Валюта, Ліміти), планувальник, меню.
* `gs_manager.py` — Клас `GoogleSheetManager`. Логіка роботи з 4-ма аркушами, очистка даних, розрахунок балансів.
* `visuals.py` — Модуль для генерації графіки (кругові діаграми).
* `config.py` — **(.gitignore)** Токени, ID таблиці, список користувачів.
* `service_account.json` — **(.gitignore)** Ключ доступу до Google Cloud API.
* `Run_Bot.bat` — Скрипт-сторож. Автозапуск, авто-оновлення (`git pull`) та рестарт.

### 📊 Структура Google Таблиці (4 аркуші)

**1. Аркуш "Транзакції" (Лог гривні)**
Бот додає нові записи у **3-й рядок**.
* **A:** Дата | **B:** Категорія | **C:** Сума (UAH) | **D:** Тип ("Витрати"/"Поповнення") | **E:** Опис | **F:** Примітка | **G:** Хто

**2. Аркуш "Валюта" (Лог долара)**
Журнал операцій з USD.
* **A:** Дата | **B:** Операція (Купив/Продав/Витратив/Отримав) | **C:** Сума ($) | **D:** Курс | **E:** Сума (грн) | **F:** Опис

**3. Аркуш "Планування" (Ліміти)**
* **A:** Категорія | **B:** Ліміт

**4. Аркуш "Налаштування" (Списки)**
* **A:** Категорії витрат | **B:** Категорії доходів

### ✅ Поточний функціонал (v3.1)

**🏦 Глобальний Гаманець:**
1.  **Баланс UAH:** Рахується як (Всі доходи - Всі витрати) за всю історію.
2.  **Баланс USD:** Окремий "сейф", який поповнюється/витрачається через меню валют.
3.  **Smart Cleaning:** Бот розуміє "брудні" числа з Excel (напр. `2 600,00 грн`) і коректно їх рахує.

**💵 Валютні операції:**
1.  **Купив $:** Списання UAH -> Зарахування USD.
2.  **Продав $:** Списання USD -> Зарахування UAH.
3.  **Витратив $:** Пряме списання USD (без впливу на гривню).
4.  **Отримав $:** Пряме зарахування USD (подарунок/зп).

**💰 Облік та Налаштування:**
1.  **Динамічні категорії:** Списки категорій завантажуються з таблиці ("Налаштування") при старті/рестарті.
2.  **Стандартні витрати:** Швидка витрата / Детальна витрата / Дохід.
3.  **Контроль:** Ліміти бюджетів, алерти при перевищенні, звіт по лімітах.

**📊 Аналітика:**
1.  **Статистика:** Показує "Всього на руках (UAH+USD)" + Потік поточного місяця.
2.  **Діаграми:** Donut chart витрат.
3.  **Звіти:** Тижневий та Місячний.
