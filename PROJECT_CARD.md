# 📂 BIMIPAL PROJECT CARD (v5.0 "Autopilot & Smart Control")

**Мета:** Накопичити кошти на відпустку під кінець літа, використовуючи автоматизацію для усунення ліні при фіксації витрат, та проактивні ліміти для фінансової дисципліни.

### 🛠 Технічний стек (Current State)
* **Мова:** Python 3.12+ (aiogram 3.x, асинхронність).
* **База даних:** Google Sheets API (gspread) + розумне кешування in-memory (60s).
* **Інтеграції:** Monobank Personal API (Webhooks) через `aiohttp` та Cloudflare Tunnel.
* **Аналітика:** Matplotlib, ReportLab.
* **Інфраструктура:** Docker на CasaOS (режим 24/7). CI/CD через `start.sh` та GitHub.
