## 📂 BIMIPAL PROJECT CARD (v3.3 Release)

**Опис:**
Telegram-бот для ведення сімейного бюджету (Вадим та Аня). Повна автоматизація фінансів, роздільні баланси, PDF-звіти та розумні нагадування. Проєкт розрахований на роботу 24/7.

### 🛠 Технічний стек
* **Мова:** Python 3.12+
* **Core:** `aiogram 3.x` (повна асинхронність)
* **Data:** `gspread` + Google Sheets API
* **Analytics:** `matplotlib` (діаграми), `reportlab` (генерація PDF)
* **Scheduling:** `apscheduler` (нагадування, крони)
* **Utils:** `html` (безпечний текст), `re` (парсинг сум)
* **Хостинг:** Локальний ПК (Windows 10/11) у режимі 24/7.

---

### 🗂 Структура файлів
* `main.py` — Точка входу. FSM, хендлери, UI/UX (клавіатури), асинхронний запуск.
* `gs_manager.py` — Backend. Клас `GoogleSheetManager`. Всі операції з таблицями, кешування, статистика.
* `reports.py` — Генератор PDF. Верстка звітів, інтеграція графіків.
* `visuals.py` — Генератор діаграм (Donut chart) за допомогою Matplotlib (Agg backend).
* `config.py` — **(.gitignore)** Токени, ID таблиці, Users ID.
* `service_account.json` — **(.gitignore)** Ключі доступу Google API.

---

### 📊 Структура Google Таблиці (5 аркушів)

1.  **Транзакції:** Основний лог (Дата | Категорія | Сума | Тип | Опис | Примітка | Хто).
2.  **Валюта:** Журнал USD операцій (Купівля/Продаж/Сейф).
3.  **Планування:** Ліміти бюджетів по категоріях.
4.  **Налаштування:** Списки категорій (динамічне підтягування).
5.  **Нагадування:** Список регулярних платежів (День | Сума | Назва | Дата ост. оплати).

### ✅ Функціонал (v3.3 Stable)

**👥 Сімейний Банкінг:**
1.  **Роздільні Гаманці:** Бот рахує особистий баланс Вадима та Ані окремо.
2.  **Перекази:** Функція `💸 Переказ` (Вадим ➡️ Аня) корегує особисті баланси без зміни загального сімейного капіталу.
3.  **Глобальний облік:** Статистика показує загальну суму UAH + USD сейф.

**📜 Повний Контроль:**
1.  **Історія:** Перегляд останніх записів з пагінацією (гортання сторінок).
2.  **Редагування:** Можливість видалити помилковий запис в один клік (з підтвердженням).
3.  **Безпека:** Екранування HTML-тегів у нотатках.

**📅 Розумний Секретар:**
1.  **Нагадування:** Щоденна перевірка регулярних платежів (Оренда, Інтернет).
2.  **Інтерактив:** Кнопка `[✅ Сплатити]` прямо в нагадуванні — автоматично створює транзакцію і оновлює дату наступної оплати.

**📊 Pro-Звітність:**
1.  **PDF-Звіти:** Генерація професійного звіту за будь-який місяць/рік.
    * Структура доходів/витрат.
    * Кругова діаграма.
    * ТОП-10 транзакцій.
2.  **Ліміти:** Візуалізація виконання бюджету (🟢/🟡/🔴).
